# Longest Valid Parentheses

This repository contains a Java solution for the Longest Valid Parentheses problem.

## Approach
- Uses a stack to keep track of indices.
- Maintains the starting index of the current valid substring.
- Updates the maximum valid parentheses length while traversing the string.

## Time Complexity
O(n)

## Space Complexity
O(n)

## Sample Input
(()

Output:
2

Input:
)()())

Output:
4

Input:
""

Output:
0
